# 前端工程師面試 Q&A

一份整理給前端工程師的面試題庫與準備筆記，內容以繁體中文撰寫，涵蓋觀念題、實戰程式碼分析與口頭回答講稿。

## 🔗 線上瀏覽

GitHub Pages：<https://tzuchianhsu.github.io/frontend-qna/>

## 內容分類

- 自我介紹與求職動機
- 經歷與行為面試題（STAR）
- 開發流程與團隊協作
- 反問公司的問題
- ⭐ 面試官最愛追問（深度題）：watch vs watchEffect、父子元件生命週期、記憶體洩漏排查、虛擬捲動、Canvas 渲染流程、協同編輯衝突（OT/CRDT）、Node.js／後端思維、AI Agent 核心流程、效能優化量化數據
- HTML / CSS / JavaScript / TypeScript
- React / Vue / 框架共通與選型
- 版本控制（Git）
- 網路與瀏覽器（HTTP、CORS、Cookie/Storage…）
- 效能優化
- 設計方法 / 設計模式
- 資料結構與演算法

## 特色

- 🔍 即時搜尋：輸入關鍵字（如 `closure`、`useEffect`、`Pinia`、`Zod`）自動篩選並展開
- 展開 / 收合全部、分類導覽列
- 可收合卡片：點題目展開答案
- 程式碼分析範例（`this` 指向、Event Loop / Promise 輸出順序…）
- 🎤 口頭講稿：深度題與行為題附約 30 秒的口語回答版本
- 列印友善（列印時自動展開全部）

## 使用方式

線上直接開上方連結即可；或下載後用瀏覽器開啟 [`index.html`](./index.html)，無需安裝任何相依套件。
